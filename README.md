# Python图书借阅推荐系统 python1122

## 项目概述

本项目是基于Python开发的图书借阅推荐系统，旨在为用户提供便捷的图书搜索、借阅及信息查阅服务。系统采用前后端分离的技术架构，后端使用Django框架，数据库采用MySQL，前端使用Vue.js。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

### 基础功能

- 图书借阅
- 图书信息查阅（最新、最热、推荐）
- 图书搜索
- 图书分类

### 管理功能

- 图书详情
- 图书信息管理
- 分类管理
- 评论管理
- 标签管理

## 系统角色

- 普通用户：可进行图书搜索、借阅、查看图书信息等操作
- 管理员：负责图书信息管理、分类管理、评论管理和标签管理等操作

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- Vue.js环境

## 部署步骤

1. 搭建Python环境，安装Django框架
2. 配置MySQL数据库
3. 部署后端服务
4. 搭建Vue.js环境，部署前端服务
5. 运行系统

- 使用Django框架，开发效率高，易于维护
- 前后端分离，便于团队协作开发
